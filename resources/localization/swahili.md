This tracker coordinates Swahili localization work for BOSC Community Library resources. It helps contributors see what has been translated, what needs review, and what should be prioritized next.

## Style Guidelines

- Use clear, widely understood Swahili suitable for learners.
- Keep technical terms consistent across files.
- Preserve original links, headings, tables, and code blocks unless a localized equivalent is needed.
- Add reviewer notes for terms that may vary by region.
