## Overview

Open data in scientific research means making research data available for inspection, reuse, and verification under clear legal and ethical conditions. It supports transparency, reproducibility, and collaboration across institutions.

## Key Practices

| Practice | Purpose |
|----------|---------|
| Use open formats | Keep data readable without proprietary tools |
| Document methods | Help others understand how data was collected |
| Add metadata | Make datasets searchable and reusable |
| Protect privacy | Remove or secure sensitive personal information |
| Choose clear licenses | Explain what reuse is allowed |

## Public Value

Open research data can improve education, public policy, environmental planning, health systems, and local innovation. It also helps communities verify claims made from publicly funded research.
