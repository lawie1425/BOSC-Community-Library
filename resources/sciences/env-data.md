## Overview

Environmental data collection gathers observations about air, water, soil, biodiversity, climate, and land use. Good collection methods make the data trustworthy, comparable, and useful for public decision-making.

## Common Methods

| Method | Example Use |
|--------|-------------|
| Field observation | Recording visible erosion or habitat change |
| Sensor monitoring | Measuring air quality, rainfall, or water levels |
| Sample testing | Analyzing soil or water quality |
| Remote sensing | Using satellite or drone imagery |
| Community reporting | Gathering local observations from residents |

## Quality Checklist

1. Define the question before collecting data.
2. Record date, location, method, and equipment.
3. Use consistent units.
4. Validate unusual readings.
5. Store raw data separately from cleaned data.
